En primer lugar se construyó una base de datos de indicadores a nivel pais para 
BOLIVIA (BOL), CHILE (CHL), ARGENTINA(ARG) Y VENEZUELA(VEN).

Los indicadores se cosntruyen de múltiples bases de datos
